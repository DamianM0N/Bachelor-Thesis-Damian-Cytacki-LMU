**English**:
For this purpose, data from various supernovae from the Supernova Cosmology Project were used to estimate these values along with their uncertainties.
The model applied is the Lambda-CDM model (the standard model of cosmology), which includes the cosmological constant.

The second part also focuses on determining the corresponding parameters, but for the DGP model, in which the cosmological constant does not appear.

Finally, a comparison of the models is performed to determine which model currently provides a better description of the data.
